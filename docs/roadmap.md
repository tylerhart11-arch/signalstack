# Roadmap

## Next 1-2 Iterations

1. Add scheduled refresh and historical backfill so charts are based on stored history instead of startup-only snapshots.
2. Expand anomaly rules into sector, factor, credit-equity lead/lag, and relative-value signals.
3. Improve thesis mapping with entity extraction, better ETF/stock ranking, and regime-aware recommendations.

## After That

1. Add editable watchlists and let thesis outputs flow directly into them.
2. Add alerting for threshold breaches in anomalies and regime shifts.
3. Add portfolio overlays so theses can be compared against current holdings.
4. Add richer charting with longer history windows, comparison views, and exportable snapshots.
